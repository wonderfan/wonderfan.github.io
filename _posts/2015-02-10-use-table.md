### Table

You can create tables by assembling a list of words and dividing them with hyphens ` - ` (for the first row), and then separating each column with a pipe ` | `:

```md
| First Header  | Second Header |
| ------------- | ------------- |
| Content Cell  | Content Cell  |
| Content Cell  | Content Cell  |
```
